# visual-qa checklist

## 1. QA Inventory

- 変更した画面
- 変更した状態
- 最終回答で主張すること
- 各主張に対応する確認手段

## 2. Core Viewports

- desktop 幅
- mobile 幅

## 3. KeepGoing Core States

### ホーム

- 個人表示
- `みんなで` 表示
- ふわふわのひとこと
- 進捗 0 / 途中 / 満タン

### セッション

- カウントダウン
- 初回ヒント
- 再生中
- 一時停止
- スキップ
- 終了確認

### メニュー / 設定

- 通常状態
- 検索 / フィルタ
- 空状態
- みんなで時の注意文

## 4. Evidence

- スクリーンショットを残す
- 再現手順を 1-3 行で残す
- 未確認項目があれば明示する
