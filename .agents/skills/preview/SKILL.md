# /preview - プレビュー確認

開発サーバーを起動して軽く画面確認する。
本格的な desktop / mobile UI QA は `/visual-qa` を使う。

## 手順

1. `npm run dev -- --host 127.0.0.1` で dev サーバーを起動
2. 必要な画面へ移動して表示を確認
3. 画面キャプチャが必要なら `playwright-interactive` か `screenshot` を使う
4. コンソールエラーや表示崩れがあれば併せて確認する
4. 結果をユーザーに共有

## 引数
- パスが指定された場合は、その画面まで移動して確認する
- `mobile` が指定された場合はモバイル幅も確認する
