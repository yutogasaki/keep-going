# ふわふわメッセージのトーンガイド

ホームのふわふわは `ほめる先生` ではなく、`また会えた相手` として話す。

## 基本原則

- `褒めない`: えらい / すごい / がんばったね / できたね を主役にしない
- `評価しない`: 多い / 少ない / 早い / 遅い / もっと を言わない
- `指示しない`: 〜してね / 〜しよう / もう一回 を言わない
- `回数に触れない`: 何回やったか、連続日数、順位づけを会話に持ち込まない
- `再会を主語にする`: また会えた、来てくれた、ぽかぽかした、気配が戻った、を優先する

## 名前呼び

- 名前は歓迎や再会の文脈でやさしく使う
- 名前で管理しない
- NG 例: `○○ちゃん、今日も3回できたね`
- OK 例: `○○ちゃん、また会えてうれしいな`

## 文脈別の使い分け

### daily

- 役割: 再会 / 気配 / 今の空気
- OK 例: `また会えたね`, `ひさしぶりだね`, `なんだか ぽかぽかしてるよ`
- NG 例: `今日もえらいね`, `昨日よりがんばったね`

### ambient

- 役割: 外の世界の気配を伝える
- OK 例: `あたらしいメニューがあるみたい`, `のぞきにいけたらたのしそう`
- NG 例: `早く見に行こう`, `これを選ぶといいよ`

### afterglow

- 役割: セッション直後の余韻を受け止める
- OK 例: `さっきのぽかぽか まだのこってるよ`, `またあえて ぽかぽかも もどってきたよ`
- NG 例: `いっぱいやったね`, `次もこのまま続けよう`

### milestone

- 役割: 出会いと変化の報告
- OK 例: `たまごが きたみたい`, `うまれたみたい`, `おおきく なったみたい`
- NG 例: `ついにレベルアップ`, `育成成功`

## みんなで！ 文脈

- `家族で来た` をうれしさとして扱う
- 人数は空気の説明にだけ使い、優劣や比較に使わない
- OK 例: `みんなで いると おちつくね`, `また みんなで あえたね`
- NG 例: `3人そろったからボーナス`, `今日は全員来たね`
